# 🏏 IPL Performance & Match Insights Analysis

> Analyzed IPL data to uncover team performance, player achievements, and match-winning patterns using Power BI and DAX.

---

## 🔥 Problem

Cricket teams and analysts need to understand:

* Which teams dominate across seasons
* What factors influence match outcomes
* Which players consistently perform

This project explores IPL data to extract meaningful insights.

---

## 📁 Dataset

* Source: IPL match & ball-by-ball dataset
* Data includes:

  * Match details (season, teams, winner)
  * Player performance (runs, centuries)
  * Ball-by-ball events

---

## 🛠 Tools Used

* Power BI → Dashboard & visualization
* DAX → Advanced calculations
* SQL → Data querying

---

## ⚙️ Key Analysis Performed

* Matches won by each team per season
* Runner-up identification logic
* Century tracking (player performance)
* Team vs team comparisons
* Season-wise trends

---

## 📈 Key Insights

* 🏆 Certain teams consistently dominate across multiple seasons
* 📊 Match outcomes strongly depend on top-order performance
* 💯 Centuries are rare but high-impact events influencing match wins
* 🔄 Team performance fluctuates significantly across seasons

---

## 💡 Insights for Strategy

* Teams should strengthen top batting order
* Player consistency is more valuable than one-time performance
* Data-driven team selection can improve win rates
